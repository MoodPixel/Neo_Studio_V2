# Assistant Project Brain

The Assistant Project Brain has four knowledge layers:

1. Stable guides from the repo `guides/` folder.
2. Live snapshots captured from the current surface through Assistant > Project.
3. Indexed project data from Neo-owned metadata sidecars and saved records.
4. User-provided project knowledge such as notes, uploaded documents, brand files, and manually saved memory.

Use stable guides for how Neo works, live snapshots for what Neo is doing now, indexed metadata for what Neo did before, and user project files for client or brand-specific constraints.
